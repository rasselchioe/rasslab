Stash
A personal finance tracker

Track income and expenses, set monthly budgets and savings goals, and see where your money goes with clean analytics. Stash handles multiple currencies with live exchange rates, and can read a bank-statement CSV to automatically detect your recurring charges and frequent stores — so subscriptions and weekly shops set themselves up, with vendor autocomplete and category auto-fill.

Frontend React TypeScript Vite Tailwind CSS Chart.js
Backend Node Express Prisma SQLite JWT auth
Tooling Playwright Render Cloudflare
